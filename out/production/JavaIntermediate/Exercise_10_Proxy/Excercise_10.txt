PROXY

Nivel: 🔥🔥

Objective:
Implement Proxy pattern according to the following statement.

Activity:
The paid Spotify service allows you to download songs to listen offline. When a user requests to download a song, the Spotify service checks if they are a premium user, to give them access to this functionality. Users have an identifier and a string with the type of user (“Free”, “Premium”).

Implement a Download service class that allows downloading a song and using the proxy pattern to check if it is possible to do so. Create the necessary test cases to ensure the quality of the download() method
